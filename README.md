# Library Management System

A web-based Library Management System developed to manage books, members, and issued records efficiently. The application performs CRUD operations and stores data in a database for easy retrieval and management.

## Features
- Manage Books, Members, and Issued Records
- Perform Create, Read, Update, and Delete (CRUD) operations
- Responsive user interface using HTML, CSS, and JavaScript
- Database integration for storing and retrieving library data

## Technologies Used
- HTML
- CSS
- JavaScript
- SQL
- Node.js
- Express.js
- SQLite
